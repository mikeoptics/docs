Joining The Grind MC is quick and easy — all you need is a legitimate copy of Minecraft Java Edition and a few minutes to get set up. This page walks you through adding the server to your server list and connecting for the first time, plus covers the most common issues players run into along the way.

## Server Details

| Detail | Info |
|---|---|
| **Server IP** | `play.thegrindmc.net` |
| **Edition** | Java Edition |
| **Recommended Version** | Latest stable release |

## How to Connect

<Steps>
  <Step title="Open Minecraft Java Edition">
    Launch the official Minecraft launcher and start **Minecraft Java Edition**. Make sure you're running the latest stable version for the best experience.
  </Step>
  <Step title="Click 'Multiplayer'">
    From the main menu, click the **Multiplayer** button to open your server list.
  </Step>
  <Step title="Click 'Add Server'">
    At the bottom of the multiplayer screen, click **Add Server** to open the server entry form.
  </Step>
  <Step title="Enter the Server IP">
    In the **Server Address** field, type `play.thegrindmc.net`. You can give the server a name like `The Grind MC` in the **Server Name** field so it's easy to find later.
  </Step>
  <Step title="Click 'Done' and Connect">
    Click **Done** to save the entry, then select the server from your list and click **Join Server**. You'll be connected and spawned in within moments.
  </Step>
</Steps>

<Note>
  Always use the **latest stable release** of Minecraft Java Edition when connecting. Snapshot or pre-release builds may not be supported and can cause unexpected connection errors. Check our Discord for the currently supported version if you're unsure.
</Note>

## Troubleshooting

Running into trouble? These are the most common connection issues players encounter and how to fix them.

<AccordionGroup>
  <Accordion title="Connection refused or timed out">
    If you see a **"Connection refused"** or **"Timed out"** error, try the following:

    - Double-check that you've entered the server IP correctly: `play.thegrindmc.net`
    - Make sure your internet connection is active and stable.
    - Temporarily disable any VPN or firewall software that might be blocking the connection.
    - The server may be undergoing brief maintenance — check the **#server-status** channel on our Discord for updates.
  </Accordion>
  <Accordion title="Outdated client / version mismatch">
    If you see a message saying your client is **outdated** or that there is a **version mismatch**, your Minecraft version doesn't match what the server is running.

    - Open the Minecraft Launcher, select the **Latest Release** profile, and launch the game.
    - If you're using a third-party launcher, make sure the version is set to the correct release.
    - Check our Discord for the exact supported version if the latest release still doesn't work.
  </Accordion>
  <Accordion title="Server full">
    If the server is at maximum capacity, you'll see a **"Server is full"** message. Here's what you can do:

    - Wait a few minutes and try connecting again — spots open up frequently.
    - Donor ranks include a **priority queue** that lets you skip the line and join even when the server is full. Check out our rank packages for more details.
    - Keep an eye on the **#server-chat** channel in our Discord to see when things quiet down.
  </Accordion>
</AccordionGroup>

## Your First Moments on the Server

Once you connect for the first time, here's what to expect:

- **You'll spawn at Spawn** — our central hub area designed to help new players get oriented.
- **Read the Welcome Board** — it's right at spawn and contains essential information about the server, rules, and key commands.
- **Claim your Starter Kit** — new players receive a starter kit with basic tools and supplies to help kick off their adventure. Run `/kit starter` to claim it.

From here, you're ready to explore, build, and grind. Welcome to The Grind MC!
