Commands are the primary way you interact with server features on The Grind MC. Whether you're navigating the world, chatting with other players, or managing your experience, every action has a command to match. This page covers all general-purpose commands available to players.

<Info>
  Commands use these conventions: `[optional]`, `<required>`. Do not include the brackets when typing commands.
</Info>

## Navigation Commands

Use these commands to move around the server quickly and manage teleport requests with other players.

| Command | Description |
|---|---|
| `/spawn` | Teleport back to the main spawn |
| `/warp [name]` | Teleport to a public warp location |
| `/warp list` | View all available warp locations |
| `/tpa [player]` | Request to teleport to another player |
| `/tpaccept` | Accept a teleport request |
| `/tpdeny` | Deny a teleport request |
| `/back` | Return to your previous location |
| `/rtp` | Random teleport to a location in the wilderness |

## Player Utility Commands

These commands help you communicate, manage your profile, and keep track of your time on the server.

| Command | Description |
|---|---|
| `/msg [player] [message]` | Send a private message |
| `/r [message]` | Reply to the last private message |
| `/ignore [player]` | Ignore a player's messages |
| `/list` | See who is currently online |
| `/ping` | Check your connection latency |
| `/playtime` | Check your total playtime on the server |
| `/seen [player]` | Check when a player was last online |
| `/nick [nickname]` | Set a custom nickname (requires Member+ or donor rank) |
| `/rules` | View the server rules |

## Social Commands

The Grind MC has a range of social features to enjoy with other players. Use these commands to connect, team up, and support the server.

| Command | Description |
|---|---|
| `/marry [player]` | Propose to another player (social feature) |
| `/party create` | Create a player party for group play |
| `/party invite [player]` | Invite someone to your party |
| `/vote` | Vote for the server to earn rewards |

## Miscellaneous Commands

Need help, want to report someone, or just looking for links? These commands have you covered.

| Command | Description |
|---|---|
| `/help` | Open the help menu |
| `/report [player] [reason]` | Report a player to staff |
| `/staff` | See which staff members are online |
| `/discord` | Get the Discord invite link |
| `/store` | Get the link to the donation store |

<Tip>
  Use **Tab** to autocomplete commands and player names in chat.
</Tip>
