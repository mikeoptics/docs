Homes and warps are your fastest way to travel around The Grind MC. You can save personal home locations to return to your base, farm, or any spot you want to revisit, while public warps take you directly to key server destinations in an instant. This page covers everything you need to know about setting homes and using warps.

## Home Commands

Use `/sethome` to save any location, then `/home` to return to it any time. The number of homes you can save depends on your rank.

| Command | Description |
|---|---|
| `/sethome [name]` | Save your current location as a home |
| `/home [name]` | Teleport to a saved home |
| `/home` | Teleport to your default home |
| `/delhome [name]` | Delete a saved home |
| `/homes` | List all your saved homes |

## Home Limits by Rank

Your rank determines how many homes you can save at once. Higher ranks and donor ranks unlock additional home slots.

| Rank | Max Homes |
|---|---|
| Default | 1 |
| Member | 3 |
| Veteran | 5 |
| Elite | 8 |
| Grinder (Donor) | 6 |
| Champion (Donor) | 9 |
| Legend (Donor) | 15 |

## Warp Commands

Public warps are server-managed destinations available to all players. Use them to jump to popular areas without needing to travel manually.

| Command | Description |
|---|---|
| `/warp [name]` | Teleport to a named public warp |
| `/warp list` | View all public warps |
| `/spawn` | Teleport to main spawn |

## Teleport Delay

To keep gameplay fair, all teleport commands have a short delay before moving you.

- There is a **3-second teleport delay** before you are moved to your destination.
- **Moving or taking damage** during the delay will cancel the teleport — you'll need to run the command again.
- **Donor ranks** may have a reduced or no teleport delay depending on their tier.

<Tip>
  Use `/back` after dying or teleporting to instantly return to where you were before. It's especially handy for retrieving your items after an unexpected death.
</Tip>
