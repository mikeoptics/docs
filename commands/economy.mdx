The Grind MC features a fully player-driven economy where you earn coins through jobs, selling items, and trading with other players. This page is your complete reference for every economy command on the server — from checking your balance to listing items on the auction house.

## Balance & Payments

Keep track of your coins and send money to other players using these commands.

| Command | Description |
|---|---|
| `/balance` (or `/bal`) | Check your current coin balance |
| `/pay [player] [amount]` | Send coins to another player |
| `/baltop` | View the wealthiest players on the server |

## Server Shop

The server shop lets you buy and sell items at fixed prices. You can access it in-world or via command.

| Command | Description |
|---|---|
| `/warp shop` | Open the server shop GUI |
| `/shop` | Alternative command for the server shop |
| `/sell hand` | Sell the item currently in your hand |
| `/sell all` | Sell all sellable items in your inventory |

## Auction House

The auction house is where players list items for sale to the entire server. Use these commands to browse listings, sell your own items, and manage your active auctions.

| Command | Description |
|---|---|
| `/ah` | Open the auction house browser |
| `/ah sell [price]` | List the item in hand for auction |
| `/ah selling` | View your currently listed items |
| `/ah expired` | Collect unsold or expired listings |
| `/ah [player]` | Browse a specific player's listings |

## Jobs

Jobs let you earn coins by doing what you already love in Minecraft. Join a job, level it up, and increase your pay rates over time.

| Command | Description |
|---|---|
| `/jobs` | Open the jobs menu |
| `/jobs join [job]` | Join a job |
| `/jobs leave [job]` | Leave a job |
| `/jobs info [job]` | View job details and pay rates |
| `/jobs stats` | View your job levels and earnings |
| `/jobs top` | View the top-earning players per job |

## Available Jobs

The following jobs are available on The Grind MC. Each one rewards you with coins for activities you'd be doing anyway.

- **Miner** — Earn money for mining ores and stone
- **Farmer** — Earn money for harvesting crops
- **Woodcutter** — Earn money for chopping trees
- **Hunter** — Earn money for killing mobs
- **Builder** — Earn money for placing blocks
- **Fisherman** — Earn money for fishing
- **Crafter** — Earn money for crafting items
